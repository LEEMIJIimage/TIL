## unix 명령어 정리


- touch-> 파일 생성
- mkdir-> 디렉토리 생성
- vi [filename] -> vim mode
- cd ~ -> desktop으로 이동
- cd .. -> 상위 폴더로 이동
- ls -> file list
- rm -> file remove






## git 사용법 정리


- git clone [repository name]
- git add [foldername]/[filename] -> folder에 file 생성
- git status 로 staging된 내용 확인
- git commit -> stage에 올라가있는 내용이 모두 커밋됨
- git push origin master ->원격 저장소에 저장
